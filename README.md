HOW TO:
=====================================================
PLAY THE GAME:

Mac/Linux/Cygwin:

    cd SketchWars
    ./run.sh

Netbeans:

    open Sketchwars/nbproject/project.xml in Netbeans
    click the run button
=====================================================
RUN THE TESTS:

Mac/Linux/Cygwin:

    cd SketchWars
    ./test.sh
=====================================================
RUN A SERVER:

Mac/Linux/Cygwin:

    cd SketchWars
    ./run-server.sh
=====================================================
RUN A CLIENT:

Mac/Linux/Cygwin:

    cd SketchWars
    ./run-client.sh
